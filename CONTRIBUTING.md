# Contributing to Payroll Engine AI Context Package

Thank you for your interest in contributing!

## Ways to Contribute

| Area | Description |
|:-----|:------------|
| **Context files** | Improvements to hand-crafted context in `context/` and `prompts/` |
| **Generator script** | Enhancements to `devops/Generate-AiContext.ps1` |
| **Examples** | Additional few-shot examples for the `context/07-examples.md` |
| **Translations** | Translated prompt files for non-English user roles |

## Reporting Issues

Use [GitHub Issues](https://github.com/Payroll-Engine/PayrollEngine.AiContext/issues) to report inaccuracies in the context files or problems with the generator script. Please include:

- A clear description of the problem
- The affected file and section
- Expected vs. actual content

## Pull Requests

Before opening a pull request, please:

1. Open an issue or discussion to align on the approach
2. Only edit hand-crafted files manually — generated files (`03`, `04`, `05`, `schemas/`) are overwritten by the generator
3. Keep commits focused and well-described

All contributions must be compatible with the [MIT License](LICENSE).

## Generated Files

The following files are generated by `devops/Generate-AiContext.ps1` and must **not** be edited manually:

- `context/03-scripting-api.md`
- `context/04-nocode-actions.md`
- `context/05-mcp-tools.md`
- `schemas/PayrollEngine.Exchange.schema.json`

To update generated content, modify the source repositories and re-run the generator.
